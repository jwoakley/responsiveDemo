fixed – this is us now

fluid – we kinda know this

fluid layouts use relative units and the layout increases/decreases with the browser window. layout breaks at small scale

responsive – GOALS

responsive web design is about using HTML and CSS to automatically resize, hide, shrink or enlarge a website to make it look good across all devices.
